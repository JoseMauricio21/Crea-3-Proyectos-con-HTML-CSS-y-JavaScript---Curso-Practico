# 🚀 **Práctica de HTML, CSS y JavaScript - Creación de 3 Proyectos**

Este repositorio contiene los recursos y el código necesario que **José Mauricio Manjarrez Reyes** ha utilizado para practicar y aprender **HTML**, **CSS** y **JavaScript** mediante la creación de tres proyectos fundamentales. Los proyectos están basados en el curso de **Jordan Alexander Cruz García**, quien es un experto en desarrollo web y publicó este curso en el canal de **freeCodeCamp en Español**.

## 📌 **Descripción del Curso**

El curso, con una duración aproximada de **2 horas**, guía paso a paso en la creación de tres proyectos clave para el desarrollo web. Gracias a estos proyectos, **José Mauricio Manjarrez Reyes** ha mejorado sus habilidades en diseño web, programación y la implementación de interactividad en páginas web.

## 🚀 **Tecnologías Utilizadas**

- **HTML**: Se utilizó para estructurar el contenido de las páginas web, proporcionando una base sólida.
- **CSS**: Se empleó para diseñar y estilizar la página, haciendo que los proyectos fueran visualmente atractivos y adaptables a dispositivos móviles.
- **JavaScript**: Se incorporó para añadir interactividad y dinamismo, haciendo que los proyectos fueran más funcionales.

## 📂 **Proyectos Incluidos**

### 1️⃣ **Barra de Navegación**
En este proyecto, **José Mauricio Manjarrez Reyes** desarrolló una **barra de navegación responsiva** con menús desplegables. La barra se adapta a diferentes tamaños de pantalla, y en dispositivos más pequeños, se convierte en un **menú lateral** para mejorar la experiencia del usuario.

### 2️⃣ **Carrusel de Imágenes (Slider)**
El segundo proyecto consistió en crear un **carrusel de imágenes** que cambia de imagen cuando el usuario interactúa con los controles de navegación. Cada imagen fue acompañada de un **título** y un **texto descriptivo**, ideal para mostrar testimonios o información relevante de manera dinámica.

### 3️⃣ **Landing Page con Modal**
En el tercer proyecto, **José Mauricio Manjarrez Reyes** construyó una **landing page** utilizando **CSS Grid**. Además, implementó un **botón modal** que abre una ventana emergente al hacer clic. Este proyecto puso en práctica HTML, CSS y JavaScript para lograr una página atractiva y funcional.
